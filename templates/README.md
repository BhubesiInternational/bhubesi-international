# Templates

Reusable document formats used across Bhubesi International so that output is consistent regardless of who — or which AI role — produces it.

## Available Templates

| Template | Use For |
|---|---|
| [`project-brief-template.md`](./project-brief-template.md) | Proposing a new project or venture (see [`../workflows/project-kickoff.md`](../workflows/project-kickoff.md)) |
| [`decision-log-template.md`](./decision-log-template.md) | Recording significant decisions (see [`../workflows/decision-making.md`](../workflows/decision-making.md)) |
| [`meeting-notes-template.md`](./meeting-notes-template.md) | Capturing meeting discussion and action items |
| [`proposal-template.md`](./proposal-template.md) | External proposals, pitches, and grant applications |
| [`financial-report-template.md`](./financial-report-template.md) | Periodic financial reporting by business units and projects |

## Usage

Copy the relevant template into the appropriate `business-units/` or `projects/` directory and fill it in — don't leave completed documents in this directory, which holds blank templates only.
